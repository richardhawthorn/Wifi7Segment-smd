### Features

- 0.56" 7 segment display, 4 character
- ESP8266 ESP7/12/12E Processor
- Wifi connected
- HT16K33 i2c LED driver
- FTDI chip built in
- TX & RX LED indicators
- 3.3V operation
- Mini USB power connector
- FTDI programming port
- Surface mount technology
